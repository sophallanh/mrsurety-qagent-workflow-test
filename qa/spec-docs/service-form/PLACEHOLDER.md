## Key Points from Service Form Doc

- Water Main Photo is **REQUIRED** — form will not submit without it
- LiDar is optional
- Pipe size determined by square footage (see `qa/spec-docs/CHRISTOPHER_TESTING_DOC.md` for table)
- Pressure reducer required if home > 5 years old
- Device source affects pricing (Contractor = $599.99 fixed; Homeowner/Insurance = $0)
