## 8 DocuSign Documents to Verify

Per Christopher's testing guide, there are **8 DocuSign documents**:

| # | Document | Sent To | Trigger |
|---|----------|---------|---------|
| 1 | Contractor Master Services Agreement | Contractor | Joins network |
| 2 | Work Order / Task Contract | Contractor | Deposit paid |
| 3 | Critical Change Order and Liability Release | Homeowner | Emergency |
| 4 | Return Service Call Work Order | Contractor | Return requested |
| 5 | Affidavit of Service | Contractor | Job complete |
| 6 | Conditional Lien Release w/ Invoice | Contractor | Docs submitted |
| 7 | Unconditional Lien Release | Contractor | Payment clears |
| 8 | Change Order | Homeowner | During job |

## Important: Unconditional Lien Release

Before signing the Unconditional Lien Release, verify that **contractor portal access is limited**.
After signing, portal access should be **restored**. This is a critical legal control.
