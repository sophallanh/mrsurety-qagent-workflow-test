# Screenshots – Email & DocuSign Triggers

## Naming Convention

`<trigger-name>_<YYYY-MM-DD>.png`

| Agent linked via referral | Agent (notification) | `email_agent-referral-link-notification.png` |

| DocuSign sent | Contractor / Technician / Homeowner | `email_docusign-sent.png` |
| DocuSign completed | All parties | `email_docusign-completed.png` |
| Payment / deposit processed | Homeowner + Admin | `email_payment-receipt.png` |
