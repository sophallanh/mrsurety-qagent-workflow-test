# Video Recording Guide – Technician Workflow

Place screen recordings for the **Technician** workflow in this folder.

## Naming Convention

`<step>_<description>_<YYYY-MM-DD>.<ext>`

## Scenarios to Record

| # | Scenario | Filename |
|---|----------|----------|
| 1 | Technician logs in and views work order dashboard | `01_technician-login-dashboard.mp4` |
| 2 | Technician views the work orders list | `02_technician-work-orders-list.mp4` |
| 3 | Technician opens a work order and reviews all job details | `03_technician-work-order-details.mp4` |
| 4 | Technician receives and opens DocuSign work order receipt email | `04_technician-docusign-email-received.mp4` |
| 5 | Technician signs the DocuSign document | `05_technician-signs-docusign.mp4` |
| 6 | Technician marks the job as complete | `06_technician-marks-job-complete.mp4` |
| 7 | Admin dashboard updates to show the job as completed | `07_admin-sees-completed-job.mp4` |

## Recording Tips

- Include the technician's email inbox showing the work order assignment email.
- For DocuSign, record the full document (all pages) before signing.
- Capture the job completion confirmation message clearly.
