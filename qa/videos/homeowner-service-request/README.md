# Video Recording Guide – Homeowner Service Request Workflow

Place screen recordings for the **Homeowner Service Request** workflow in this folder.

## Naming Convention

`<step>_<description>_<YYYY-MM-DD>.<ext>`

## Scenarios to Record

| # | Scenario | Filename |
|---|----------|----------|
| 1 | Homeowner logs in and starts a new service request | `01_homeowner-login-new-request.mp4` |
| 2 | Homeowner fills in property address and permit type | `02_form-address-permit-type.mp4` |
| 3 | Homeowner enters agent email in the insurance section (Method 2) | `03_agent-email-entered.mp4` |
| 4 | Homeowner submits the form and sees confirmation | `04_form-submitted-confirmation.mp4` |
| 5 | Admin sees and approves the pending agent (Method 2) | `05_admin-approves-pending-agent.mp4` |
| 6 | Homeowner submits a second request with a different address | `06_second-address-request.mp4` |
| 7 | Homeowner submits a request with no agent email (edge case) | `07_no-agent-email-edge-case.mp4` |

## Recording Tips

- Scroll slowly through the form so all fields are visible.
- Show the confirmation screen clearly after each submission.
- For Method 2, include a screen of the agent's inbox showing the welcome email received.
