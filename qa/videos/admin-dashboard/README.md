# Video Recording Guide – Admin Dashboard Workflow

Place screen recordings for the **Admin Dashboard** workflow in this folder.

## Naming Convention

`<step>_<description>_<YYYY-MM-DD>.<ext>`

## Scenarios to Record

| # | Scenario | Filename |
|---|----------|----------|
| 1 | Admin logs in (admin@mrsurety.com) and views dashboard overview | `01_admin-login-dashboard.mp4` |
| 2 | Admin reviews and approves a pending contractor registration | `02_admin-approves-contractor.mp4` |
| 3 | Admin reviews and approves a pending agent link (Method 2) | `03_admin-approves-pending-agent.mp4` |
| 4 | Admin views the jobs list and tracks job status | `04_admin-jobs-list-tracking.mp4` |
| 5 | Admin opens a job and views full detail (agent, homeowner, contractor) | `05_admin-job-detail.mp4` |
| 6 | Admin approves a work order to trigger DocuSign | `06_admin-approves-work-order.mp4` |
| 7 | Admin views completed DocuSign documents | `07_admin-views-docusign-docs.mp4` |
| 8 | Admin reviews payment/deposit status | `08_admin-payment-review.mp4` |

## Recording Tips

- Show the admin email (`admin@mrsurety.com`) at login so it's clear which account is used.
- For contractor/agent approvals, include the email notification confirmation after approval.
- Show the agent–homeowner link visible in the jobs list.
