# Videos – QA Screen Recordings

This folder contains screen recording videos organized by workflow. Each subfolder corresponds to a major workflow in the MrSurety platform.

## Folder Structure

```
videos/
├── agent-referral-workflow/       # Agent login → referral link → homeowner sign-up
├── homeowner-service-request/     # Homeowner fills and submits service request form
├── contractor-bidding/            # Contractor bids, homeowner selects, deposit flow
├── admin-dashboard/               # Admin approval, job tracking, user management
├── technician-workflow/           # Technician work order receipt and sign-off
├── email-docusign-triggers/       # Email inbox and DocuSign document verification
└── agent-upload-invite/           # Contractor invite → agent upload → revoke flow
```

## Naming Convention

`<workflow-step>_<YYYY-MM-DD>.<ext>`

**Examples:**
- `agent-referral-link-full-flow_2026-03-13.mp4`
- `homeowner-service-request-form_2026-03-13.webm`
- `contractor-bid-and-deposit_2026-03-13.mp4`

## Recording Guidelines

- Record at full resolution (1920×1080 or Retina scale)
- Keep recordings focused on one workflow or scenario per file
- Narrate steps if possible (audio optional)
- Trim dead time before/after the test scenario
- Recommended tools: QuickTime (Mac), Loom, OBS Studio
- Upload to Google Drive in this folder structure and share the link with Chris
