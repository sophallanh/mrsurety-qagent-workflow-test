# Video Recording Guide – Agent Referral Workflow

Place screen recordings for the **Agent Referral Workflow** in this folder.

## Naming Convention

`<step>_<description>_<YYYY-MM-DD>.<ext>`

**Examples:**
- `01_agent-login-referral-link_2026-03-13.mp4`
- `02_homeowner-follows-referral-link_2026-03-13.mp4`

## Scenarios to Record

| # | Scenario | Filename |
|---|----------|----------|
| 1 | Agent logs in and navigates to the Referral section | `01_agent-login-referral-section.mp4` |
| 2 | Agent generates a referral code/link | `02_referral-link-generated.mp4` |
| 3 | Homeowner opens referral link in a new/incognito browser | `03_homeowner-opens-referral-link.mp4` |
| 4 | Homeowner fills out and submits service request form | `04_homeowner-submits-service-request.mp4` |
| 5 | Agent's dashboard updates to show the linked homeowner | `05_agent-dashboard-linked-homeowner.mp4` |
| 6 | Admin confirms the agent–homeowner link | `06_admin-confirms-agent-link.mp4` |
| 7 | Attempt to reuse the same referral link (should be rejected) | `07_referral-link-reuse-rejected.mp4` |
| 8 | Agent generates a second referral link for a new request | `08_second-referral-link-generated.mp4` |

## Recording Tips

- Show the full URL bar so the referral code/token is visible.
- When testing the reuse scenario, include both the agent and homeowner browser windows.
- Narrate each step if possible.
- Trim dead time before/after each scenario.
