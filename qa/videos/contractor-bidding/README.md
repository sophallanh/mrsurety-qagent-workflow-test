# Video Recording Guide – Contractor Bidding Workflow

Place screen recordings for the **Contractor Bidding** workflow in this folder.

## Naming Convention

`<step>_<description>_<YYYY-MM-DD>.<ext>`

## Scenarios to Record

| # | Scenario | Filename |
|---|----------|----------|
| 1 | Contractor logs in and views available jobs | `01_contractor-login-available-jobs.mp4` |
| 2 | Contractor opens a job and views details | `02_contractor-opens-job-detail.mp4` |
| 3 | Contractor fills in and submits a bid | `03_contractor-submits-bid.mp4` |
| 4 | Contractor uploads an estimate document (PDF) | `04_contractor-uploads-estimate.mp4` |
| 5 | Homeowner logs in, reviews bids, and selects a contractor | `05_homeowner-selects-bid.mp4` |
| 6 | Homeowner completes the deposit payment | `06_homeowner-completes-deposit.mp4` |
| 7 | Homeowner selects a service date from the calendar | `07_homeowner-selects-date.mp4` |
| 8 | Contractor receives notification that their bid was selected | `08_contractor-bid-selected-notification.mp4` |
| 9 | Second contractor bids on the same job (competing bid scenario) | `09_competing-bid-scenario.mp4` |

## Recording Tips

- Include the estimate PDF being dragged or selected in the file picker.
- Capture the deposit form and confirmation screen clearly.
- For the calendar date selection, show the full calendar UI.
